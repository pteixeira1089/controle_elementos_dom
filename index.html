Cadastrar Funcionários

O objetivo dessa seção é integrar o script criado na atividade 10 a uma página HTML, com controle de elementos do DOM (Document Object Model).

Ao preencher o formulário ao lado, os elementos serão controlados via JavaScript com as seguintes regras:

  • Não serão aceitos registros com os campos em branco;
  • Não serão aceitos registros com valores de idade que não sejam números inteiros maiores ou iguais a 16
  • Caso se opte por cadastrar um gerente ou um desenvolvedor, o campo cargo será auotomaticamente preenchido com o cargo correspondente.

Ao cadastrar um funcionário corretamente, o script cria um objeto que o representa, armazenando-o em um array.

Por serem objetos instanciados a partir de uma classe, é possível executar seus métodos correspondentes na seção Funcionários Cadastrados

Funcionários Cadastrados

Todos os funcionários cadastrados, incluindo gerentes e desenvolvedores, podem se apresentar e trabalhar, pois estes são métodos da classe 'Funcionários', que é a mais abstrata de todas.

Funcionários que ocupam cargo de gerência são objetos de uma classe que herda da classe 'Funcionários'. Com isso, é possível definir métodos e atributos próprios a eles. Nesse sentido, gerentes possuem o atributo 'departamento', presente somente nesta subclasse, assim como o método 'gerenciar'.

De forma análoga, funcionários que ocupam o cargo de desenvovedores também são objetos de uma classe herdeira da classe 'Funcionários', possuindo o atributo 'linguagem' e o método 'programar', próprios dessa subclasse.
